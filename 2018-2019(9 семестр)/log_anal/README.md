Директория используется для анализа лога и выделения признаков.

В корне существует папка logs(вам её надо создать самим), в которой находятся сырые логи. С помощью программы filling_the_database.py лог записывается в базу данных. 

Структуру таблицы можно увидеть в ./analyse/models.py. Далее лог анализируется с помощью analyse.py, создается папка ./users/ куда сливаются данные каждого пользователя.


Примечание:
для запуска *.py  в shell можно использовать команду "exec(open(filename).read())"
